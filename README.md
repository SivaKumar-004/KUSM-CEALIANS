# Fincare AI - Proactive Financial Agent Simulator

Welcome to the **Fincare AI** interactive dashboard. This project is a modern, proactive financial technology simulation powered by **LangGraph** and **FastAPI**. It transforms traditional passive AI recommendations into automated, proactive financial interventions based on a user's calculated financial health score.

## 🚀 Features

### Core Agent Capabilities
The system leverages LangGraph to route users into one of four distinct, highly customized tracks based on their financial score. Each track triggers specific, immediately actionable interventions:

1. **💎 Resilience Builder (Score >= 75)**
   - **STABLE Path**: Auto-enrollment in Government Schemes, automated SIP setups, Top-Up Health cover analysis, weekly actionable email tips, and automated expense reviews.
2. **🛡️ Stability Anchor (Score 55 - 74)**
   - **WATCH LIST Path**: High EMI restructuring, urgent protection alerts, micro-learning debt videos, counselor alerts, and 7-day budgeting challenges.
3. **🛟 Safety Net (Score 35 - 54)**
   - **VULNERABLE Path**: Emergency fund tracker initialization, direct debt helpline routing, welfare scheme matching, and active payment deferment options.
4. **🚨 Debt Revival (Score < 35)**
   - **DEBT STRESSED Path**: Computes immediate One-Time Settlement (OTS) proposals, blocks new loan requests, simulates debt consolidation, and connects to debt management services.

### Premium Frontend & UI
The dashboard features an **Apple-esque**, premium fintech aesthetic built completely with Vanilla HTML, CSS, and JS.
- **Deep Glassmorphism**: Cards and sidebars employ heavy translucent backdrop blurs and structural inner borders.
- **Animated Gradients**: The background features a slowly shifting, living gradient mesh.
- **Interactive UI**: Contains an interactive sidebar testimonial, dynamic tabs for an Activity Log versus pending Action Items, and a fully functional "Leave a Review" interactive star-rating modal.
- **Score-Isolation**: The UI calculates the score live and strictly renders only the single, heavily customized path card matching the user's specific financial situation.

## 🛠️ Tech Stack
- **Backend**: Python, FastAPI, Uvicorn, LangGraph, LangChain, Pydantic
- **Frontend**: Vanilla HTML5, CSS3, JavaScript (Fetch API) 

## ⚙️ Installation & Setup

1. **Clone the repository and navigate to the project folder:**
   ```bash
   cd "financial_agent"
   ```

2. **Install all required dependencies:**
   Make sure you have an active Python environment.
   ```bash
   pip install -r requirements.txt
   ```

3. **Start the FastAPI Development Server:**
   ```bash
   python server.py
   ```
   *Note: The server runs on `uvicorn` and listens on port `8000` by default.*

4. **Access the Dashboard:**
   Open your browser and navigate to:
   [http://localhost:8000](http://localhost:8000)

## 🎮 How to Use the Simulator
1. Open the dashboard at `http://localhost:8000`.
2. In the **Agent Simulator** panel, adjust the inputs (Age, Score, Retirement Gap, Monthly Income, Monthly EMI).
3. Click **Run Analysis**.
4. Observe the interface adapt! Based on the **Score** parameter, you will see a unique mapped path taking up the viewing space, alongside the specifically tailored financial interventions generated by LangGraph.
5. Interact with the **Action Items** (toggling schemes, clicking Draft Proposals) and check the **System Activity Log** to see simulated background agent activities dynamically populate.
6. Click **Leave a Review** in the sidebar to test out the interactive glassmorphic modal.

## 📄 Repository Structure
- `main.py` / `graph.py` / `node.py` / `state.py`: Core LangGraph agent definitions, states, and logic routing.
- `server.py`: The FastAPI backend handling logic endpoints and serving static files.
- `templates/index.html`: The core markup structure for the comprehensive UI.
- `static/style.css`: The massive premium UI styling sheet containing animations, glassmorphism, layouts, and gradients.
- `static/app.js`: Client-side logic controlling paths, API fetching, tab-switching, toasts, and modal interactions.
